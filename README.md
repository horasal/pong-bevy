# Pong in Bevy

A simple [Pong](https://en.wikipedia.org/wiki/Pong) implementation with the [Bevy Game Engine](https://github.com/bevyengine/bevy).

![Demo](screenshot.jpg "Screenshot")

# Compile

```rust
cargo build --release
```

# How to play

Left Paddle:

* W: move up
* S: move down
* Q: manual/auto mode

Right Paddle:

* Up Arrow: move up
* Down Arrow: move down
* P: manual/auto mode
